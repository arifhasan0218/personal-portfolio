My Skills

Hyper Text Markup Lang-

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the World Wide Web (web pages). HTML uses various tags and elements to define the different parts of a web page, such as headings, paragraphs, images, links, and more. Browsers interpret the HTML code to display the content and layout of a website for users to view on their devices.

Cascading Style Sheets

CSS stands for Cascading Style Sheets. It is a stylesheet language used to control the presentation and layout of HTML documents. In simple terms, CSS allows you to define how your web page looks, including the colors, fonts, spacing, and positioning of elements. It helps separate the content from its presentation, making it easier to maintain and update the visual aspects of a website.

BOOOSTRAP 5

Bootstrap is a popular front-end framework for building responsive and mobile-friendly websites and web applications. It provides a set of pre-designed HTML, CSS, and JavaScript components that help developers create consistent and visually appealing user interfaces quickly. In short, Bootstrap simplifies web development by offering ready-to-use design elements and layouts, saving time and effort in building modern and responsive websites.

JavaScript

JavaScript is a high-level programming language used primarily for web development. It enables interactivity on websites by allowing you to add dynamic and interactive elements such as animations, form validation, and user interactions. JavaScript runs on the client-side, meaning it is executed on the user's web browser, making web pages more engaging and responsive.

Photoshop

Photoshop is graphics editing software used for image manipulation, photo editing, and graphic design, widely used by professionals for creative tasks.

Illustrator

Illustrator is vector graphics editing software used for creating and editing illustrations, logos, icons, and other scalable graphics. It is widely used by designers and artists for its precision and ability to create artwork that can be resized without losing quality.
